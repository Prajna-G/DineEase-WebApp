# 🍽️ DineEase

A responsive restaurant website built using HTML, CSS, Bootstrap, and JavaScript.

## Features

- Responsive landing page
- Restaurant reservation interface
- User login and sign-up pages
- Gallery section
- Client-side form validation

## Technologies

- HTML5
- CSS3
- Bootstrap
- JavaScript
